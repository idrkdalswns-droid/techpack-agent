# 👗 의류 테크팩 설계 에이전트 (Techpack Agent)

> 막연한 아이디어 한마디로, 공장 납품 가능한 테크팩(Tech Pack)을 완성하는 Claude AI 스킬

## 이 스킬은 뭔가요?

"여름 원피스 하나 만들고 싶어"라고 말하기만 하면, AI가 10년 경력의 패션 테크니컬 디자이너처럼 단계별로 질문을 던지며 핏, 소재, 색상, 봉제 디테일, 사이즈 그레이딩, 부자재까지 모든 변수를 잡아줍니다. 최종 결과물은 **공장에 바로 넘길 수 있는 수준의 테크팩 Word 문서(.docx)**로 출력됩니다.

## 주요 기능

- **7단계 인터뷰 워크플로우**: 아이템 정의 → 실루엣 & 핏 → 소재 선택 → 색상 & 염색 → 사이즈 & 그레이딩 → 봉제 & 부자재 → 라벨링 & 패키징
- **모든 의류 카테고리 지원**: 원피스, 티셔츠, 데님, 셔츠, 아우터, 이너웨어 등
- **전문 지식 탑재**: 2025-2026 S/S 트렌드, 소재 공학, 그레이딩 규칙, 염색 로트 관리, 친환경 워싱 기술 등
- **한국어 중심 (영문 전문용어 병기)**: 국내 공장 커뮤니케이션에 최적화
- **"모르겠다" 대응**: 사용자가 모르는 항목은 자동 추천 후 확인

## 설치 방법

### Claude.ai (웹/앱)

1. 이 레포지토리의 `techpack-agent` 폴더를 다운로드합니다
2. 폴더를 `.zip`으로 압축합니다
3. Claude.ai → **설정(Settings) → 기능(Capabilities) → Skills**에서 업로드합니다
4. **코드 실행(Code Execution)**이 활성화되어 있는지 확인합니다

### Claude Code (터미널)

```bash
# 개인 스킬로 설치
git clone https://github.com/YOUR_USERNAME/techpack-agent.git
cp -r techpack-agent/techpack-agent ~/.claude/skills/

# 또는 프로젝트별 설치
cp -r techpack-agent/techpack-agent .claude/skills/
```

## 사용법

별도 명령어 없이, 의류 제작 관련 대화를 시작하면 자동으로 작동합니다.

**예시:**
- "여름 원피스 만들고 싶어"
- "데님 바지 기획해줘"
- "남녀공용 후드 테크팩 만들어줘"
- "브랜드 런칭 준비 중인데 공장에 보낼 서류 필요해"
- "겨울 아우터 하나 기획해볼까"

```
techpack-agent/

└── references/
    └── fashion-knowledge-base.md         # 의류 전문 지식 베이스
```

## 테크팩 출력 구조

생성되는 테크팩 Word 문서에는 다음 섹션이 포함됩니다:

| 섹션 | 내용 |
|------|------|
| 커버 시트 | 브랜드, 시즌, 스타일 번호, 사이즈 범위 |
| 기술 디자인 설명 | 앞면/뒷면 구조, 디테일 콜아웃, 색상 |
| 자재 명세서 (BOM) | 겉감, 안감, 심지, 봉제실, 부자재 전체 |
| 측정 사양서 (Spec Sheet) | POM 치수, 그레이딩, 허용 오차 |
| 봉제 사양서 | 솔기 처리, 스티치, 시접, 보강 봉제 |
| 색상 & 가공 사양서 | 염료, QC 기준, 워싱 지시 |
| 라벨링 & 패키징 | 라벨, 케어 심볼, 폴딩, 포장재 |
