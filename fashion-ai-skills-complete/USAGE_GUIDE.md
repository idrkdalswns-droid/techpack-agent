## 2. 스킬 1: 프롬프트 아키텍트

### 기본 사용법

그냥 자연스럽게 원하는 이미지를 설명하세요. 에이전트가 알아서 태그 수프를 걸러내고 역질문을 던집니다.

**사용 예시:**
```
"찰랑거리는 핑크색 여름 원피스 입은 여자 화보 이미지 프롬프트 만들어줘"
```

**에이전트 반응:**
1. 즉시 렌더링 거부(Hold)
2. 결측치 파악 후 역질문:
   - "원단이 시폰인가요, 리넨인가요? (빛 투과율이 달라져요)"
   - "촬영 목적이 이커머스 상세컷인가요, 감성 화보인가요?"
   - "배경은 스튜디오 흰 배경? 야외?"
3. 답변 수집 후 5단계 구조 프롬프트 + JSON 슈도코드 출력

### 출력물 활용법

에이전트가 두 가지를 출력합니다:

**① 영문 통합 프롬프트** → 나노바나나 프로에 바로 복사-붙여넣기
**② JSON 슈도코드** → 변수만 교체하면 시리즈 이미지 양산 가능

### 핵심 팁

- "알아서 해줘"라고 하면 최적값을 추천하고 확인을 받습니다
- 테크팩이 이미 있으면 "이 테크팩의 아이템으로 프롬프트 만들어줘"라고 하세요
- 원단을 말하면 자동으로 물리적 비중 지표가 삽입됩니다 (시폰 → fluttering, liquid motion 등)

## 3. 스킬 2: 버추얼 스튜디오 디렉터

### 기본 사용법

촬영 목적을 말하면 카메라, 조명, 레퍼런스 슬롯을 자동으로 세팅합니다.

**사용 예시:**
```
"이 원피스로 Vogue 스타일 에디토리얼 화보 찍을 건데 세팅해줘"
```

**에이전트 반응:**
1. 촬영 목적 확인 (에디토리얼)
2. 14슬롯 레퍼런스 배정 전략 제안
3. 조명 세팅 자동 선택 (Rembrandt 조명)
4. 카메라/렌즈 추천 (Hasselblad X2D + 135mm)
5. 물리 연산 체크리스트 확인
6. 최종 프롬프트 출력

### 레퍼런스 이미지 활용법

나노바나나 프로는 최대 14장의 참조 이미지를 동시 처리합니다. 에이전트가 슬롯 배분을 가이드합니다:

```
"이 모델 사진 3장, 이 옷 사진 2장, 이 분위기 사진 5장을 레퍼런스로 쓸 거야"
→ 에이전트가 슬롯 1~3: Identity Fix / 4~5: Garment Fix / 6~10: Mood Fix 배정
```

### 시리즈 촬영

한 번의 세팅으로 여러 컷을 만들 수 있습니다:

```
"같은 모델, 같은 옷으로 5가지 다른 배경에서 찍고 싶어"
→ 환경/조명만 변경된 5개 프롬프트를 한 번에 출력
```

## 4. 스킬 3: VTO 에디터 & 스케일러

### 인컨텍스트 편집 사용법

이미 생성된 이미지의 일부분만 수정할 때 사용합니다.

**사용 예시:**
```
"이 사진에서 셔츠만 청키한 니트로 바꿔줘"
```

**에이전트 반응:**
1. 변경 타깃 식별 (셔츠)
2. 보존 방벽 자동 생성 (얼굴, 조명, 배경, 포즈)
3. 편집 프롬프트 출력 (영문)

**핵심:** 바꿀 것만 말하면 에이전트가 알아서 "바꾸면 안 되는 것" 리스트를 만들어 프롬프트에 넣어줍니다.

### 가상 피팅(VTO) 사용법

**준비물:** 모델 사진 1장 + 의상 사진 1장

```
"이 모델 사진에 이 옷 사진의 의상을 입혀줘"
```

에이전트가 3단계 파이프라인을 안내합니다:
1. 이미지 업로드 가이드
2. 피팅 프롬프트 자동 생성
3. 동일 세팅에서 추가 의상 교체 제안

**모델 사진 없을 때:**
```
"모델은 없고 옷 사진만 있어"
→ 에이전트가 가상 모델 생성 프롬프트를 먼저 만들어 드립니다
```

### 배경 교체 사용법

```
"이 화보를 눈 덮인 산으로 배경 바꿔줘"
```

에이전트가 단순 누끼가 아니라 환경광 재계산까지 포함된 프롬프트를 생성합니다.

### 텍스트/로고 삽입

```
"이 룩북 위에 '2026 SUMMER COLLECTION' 텍스트 넣어줘"
```

에이전트가 오클루전(모델 뒤로 텍스트가 가려지는) 효과까지 설계합니다.

### 대량 양산 세팅

```
"이커머스 제품 이미지 300장 자동 생성할 수 있게 API 세팅해줘"
```

에이전트가 안내하는 것:
1. 히어로 컷 프롬프트 템플릿 설계
2. 변수 슬롯 정의 (엑셀 연동 가능)
3. 해상도 분리 발주 전략 (웹=2K, 인쇄=4K)
4. 비용 최적화 가이드

## 5. 실전 시나리오별 가이드

### 시나리오 A: 테크팩 완성 → 첫 번째 룩북 이미지

1. **스킬 1**에게: "이 테크팩의 더스트 로즈 원피스로 에디토리얼 화보 프롬프트 만들어줘"
2. **스킬 2**에게: "Golden Hour 야외 촬영으로 세팅해줘"
3. 나노바나나 프로에서 이미지 생성
4. **스킬 3**에게: "배경을 지중해 거리로 바꿔줘" (필요 시)

### 시나리오 B: 이커머스 제품 촬영 대량 양산

1. **스킬 1**에게: "흰 배경 이커머스 제품 사진 프롬프트 템플릿 만들어줘"
2. **스킬 2**에게: "High-key 상업 조명으로 세팅"
3. **스킬 3**에게: "이 템플릿으로 제품 50종 배치 렌더링 세팅해줘"

### 시나리오 C: 한 벌의 옷으로 10컷 룩북

1. **스킬 1** + **스킬 2**로 첫 번째 이미지 생성
2. **스킬 3**에게: "같은 옷, 같은 모델로 배경만 10가지로 바꿔줘"
3. **스킬 3**에게: "각 이미지에 브랜드 로고 넣어줘"

### 시나리오 D: 평면 컷(Flat Lay)에서 착장 화보로

1. 옷 사진(Flat Lay)만 준비
2. **스킬 3 VTO**에게: "이 옷을 아시아계 20대 여성 모델에 입혀줘"
3. **스킬 3**에게: "같은 모델로 가죽 재킷, 실크 드레스, 블레이저도 순서대로 입혀줘"
4. **스킬 3**에게: "전부 인스타그램 4:5 비율로 변환해줘"

## 7. FAQ

**Q: 3개 스킬을 다 설치해야 하나요?**
A: 아니요. 필요한 것만 설치해도 됩니다. 다만 1→2→3 순서의 워크플로우를 타면 가장 좋은 결과를 얻을 수 있습니다.

**Q: 나노바나나 프로 없이도 쓸 수 있나요?**
A: 프롬프트 구조 자체는 다른 AI 이미지 생성기에서도 활용 가능하지만, 추론 모드, 14슬롯 레퍼런스, 인컨텍스트 편집 등은 나노바나나 프로의 고유 기능입니다.

**Q: 테크팩 스킬과 연동되나요?**
A: 직접적으로 연동되지는 않지만, 테크팩 스킬로 확정된 아이템 정보(원단, 색상, 핏 등)를 프롬프트 아키텍트에게 전달하면 해당 사양에 맞는 프롬프트를 설계해줍니다.

**Q: 프롬프트를 영어로 출력하는 이유는?**
A: 나노바나나 프로(Gemini 기반)는 영문 프롬프트에서 가장 높은 정확도를 보입니다. 한국어로 대화하되, 최종 프롬프트는 영문으로 출력하여 바로 사용할 수 있게 합니다.
